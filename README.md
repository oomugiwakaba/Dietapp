# Dietapp
 
Dietappは日々のカロリー等を簡潔にメモ出来ます。
 
# DEMO
 
 https://gyazo.com/aa91412235b79754faa906ad3426ba37
 
# 特徴
 
栄養価や食材などは、健康に関係しても痩せる事とは無関係と言っても過言ではありません。糖と油と体重だけ記録を付けていけば、人間は必ず痩せます。痩せる事だけに特化したダイエットサポートアプリです。
